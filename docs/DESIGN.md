# Design

## Experience Principles

- Make common workflows obvious and efficient.
- Prefer clear states over explanatory copy.
- Keep visual systems consistent across surfaces.

## Interface Inventory

Track important screens, components, and interaction states here.

- **Home** — URL entry point, live specimen preview, example chips, and generation form.
- **Docs** — Persistent top-nav destination explaining llms.txt basics, quickstart usage, domain proxy setup, and product features for developers.
- **Result** — Hosted llms.txt preview, copy/download actions, monitoring toggle, page inventory, and version history.
